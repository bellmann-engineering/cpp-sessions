# Übung: C vs. C++ – Reflexion

**Ziel:** Denken Sie über typische C-Fallstricke nach und wie C++-Features diese adressieren.

## Aufgabenstellung

1. Nennen Sie drei Probleme, die Ihnen in C-Code häufig begegnen (z.B. Speicherlecks, fehlende Typsicherheit, manuelle Ressourcenverwaltung).
2. Für jedes Problem: Welches C++-Feature könnte helfen? (z.B. RAII, Smart Pointer, Referenzen, Klassen, `enum class`).
3. Diskutieren Sie in der Gruppe: Welche dieser Features könnten Sie in Ihrem aktuellen Projekt einführen? Wo sehen Sie Hindernisse?

*Hinweis: Dies ist eine Reflexionsübung – es gibt keine Programmieraufgabe.*
