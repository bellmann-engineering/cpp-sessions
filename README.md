# C++ Embedded Training Sessions

Structured training material for Modern C++ (C++11–C++23)  
with focus on Embedded Systems and clean architecture.

## Structure

sessions/
- session-1/  → Modern C++ Basics (C++11)
- session-2/  → RAII and Resource Management
- session-3/  → Templates and Standard Library
- session-4/  → Modern Features (C++17–23)
- session-5/  → Concurrency, Ranges, Concepts
- session-6/  → Architecture, Clean Code, Framework Alignment

Each session contains:
- demos/       → Example code shown during training
- exercises/   → Practice tasks (Markdown / incomplete code)
- solutions/   → Reference implementations
